# Project 018: Sort a List

## Description
This project asks the user to enter multiple numbers, then sorts them in ascending order.  
It teaches beginners how to work with lists, loops, and Python's built-in sorting methods.

## Skills Learned
- Lists  
- Sorting lists  
- User input  
- Built-in functions  

## How to Run
1. Open `sort_list.ipynb` in Jupyter Notebook.
2. Run all cells.
3. Enter numbers separated by commas to see the sorted list.
